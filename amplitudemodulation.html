Write a matlab code for Amplitude Modulation of message signal using carrier signal


Matlab Code :

% Amplitude Modulation
clear all;
close all;
clc;
t=0:0.001:5;
fm=input('Enter the frequency of message signal : ');
fc=input('Enter the frequency of carrier signal : ');
am=input('Enter the amplitude of message signal : ');
ac=input('Enter the amplitude of carrier signal : ');
mt=am*sin(2*pi*fm*t);
ct=ac*sin(2*pi*fc*t);
st=ct+(mt.*(sin(2*pi*fc*t)));

figure(1)
subplot(3,1,1);
plot(t,mt);
xlabel('Time');
ylabel('m(t)');
title('Message Signal');

subplot(3,1,2);
plot(t,ct);
xlabel('Time');
ylabel('c(t)');
title('Carrier Signal');

subplot(3,1,3);
plot(t,st);
xlabel('Time');
ylabel('s(t)');
title('Modulated Signal');
